# Poznámka: design-inspirace.png

Do téhle složky patří screenshot moderního osobního webu, který se vám líbí.

Agent ho použije jako vizuální inspiraci pro stránku, kterou vytvoří.

## Co vybrat:
- Čistý, moderní design s hodně bílého prostoru
- Osobní web nebo portfolio — ne firemní stránka
- Ideální: screenshot z Dribbble, Awwwards, nebo web někoho, koho obdivujete

## Jak přidat:
1. Udělejte screenshot webu (Cmd+Shift+4 na Macu)
2. Uložte sem jako `design-inspirace.png`
