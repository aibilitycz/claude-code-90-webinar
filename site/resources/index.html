FAQ

Časté otázky

Jsou moje data v bezpečí?

Agent má přístup jen ke složce, ve které běží — ne k celému počítači. U důležitých akcí (mazání, instalace) se vás vždy zeptá na povolení. Vaše prompty a soubory se posílají na servery Anthropic ke zpracování. U firemních plánů (Team, Enterprise) se data nepoužívají k trénování a uchovávají se maximálně 30 dní. U osobních plánů (Pro, Max) záleží na vašem nastavení soukromí v claude.ai/settings. Pro firmy Anthropic nabízí enterprise licencování s přísnějšími zárukami.

Podrobněji: Security · Data usage

Jak šetřit tokeny?

Tokeny jsou jednotky „myšlení", které agent spotřebovává. Tři způsoby, jak je šetřit: 1) Začínejte ve správné složce — nespouštějte agenta v kořenovém adresáři. 2) Vytvořte si CLAUDE.md — jedna věta o projektu ušetří stovky tokenů. 3) Používejte /compact — zkrátí konverzaci a uvolní kontextové okno.

Podrobněji: Costs · Best practices

Co je „lokální" vs „cloud"?

Lokální — agent pracuje přímo s vašimi soubory na počítači. Výpočty probíhají na serverech Anthropic, ale soubory zůstávají u vás. Cloud — agent běží v izolovaném prostředí na serverech Anthropic, pracuje primárně s repozitáři na GitHubu. Pro začátek volte lokální.

Podrobněji: Claude Code on the web

Jaký je rozdíl mezi skills, agenty a subagenty?

Skill je textový soubor s instrukcemi — říká agentovi, jak provést konkrétní úkol. Načítá se na vyžádání. Subagent je izolovaný pracovník s vlastním kontextovým oknem — agent si ho pouští na dílčí úkoly, aby nezaplnil svůj hlavní kontext. Plugin je balíček, který může obsahovat skills, subagenty, hooky a MCP servery dohromady.

Podrobněji: Skills · Features overview

Jak se vrátit zpátky, když agent něco pokazí?

Dvakrát Escape (nebo příkaz /rewind) otevře nabídku, kde vrátíte kód, konverzaci, nebo oboje na libovolný předchozí bod. Jedním Escape zastavíte agenta uprostřed práce. Nebo prostě řekněte: „Vrať to zpátky."
