Doporučené čtení

Jak pracovat s agenty lépe

  • Harness Engineering — Martin Fowler

    Jak budovat důvěru v AI agenty. Agent = model + harness. CLAUDE.md, testy a nástroje jako feedforward a feedback kontroly.

  • Harness Engineering — Latent Space

    Praktický pohled na to, jak strukturovat kód, dokumentaci a kontext pro autonomní agenty. Composable skills, observability, CLI-first přístup.

  • Claude Skills are awesome — Simon Willison

    Proč jsou skills možná důležitější než MCP. Od autora Djanga a jednoho z nejstřízlivějších hlasů v AI.

  • Compound Engineering — Every

    Jak pracovat s agenty tak, aby se každý dokončený úkol stal investicí do dalšího. Plan → work → review → compound.
