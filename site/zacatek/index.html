Začátek

Nainstalujte si Claude Code a napište první zprávu

Claude Code je AI agent od Anthropic. Nejen že odpovídá na otázky — umí pracovat s vašimi soubory, hledat na webu, stavět weby a aplikace, a automatizovat opakovanou práci. Tahle stránka vás provede od nuly k prvnímu rozhovoru.

Asi 15 minut Pro každého
Co to je

Claude Code v jedné větě

Je to AI agent, kterému řeknete, co potřebujete, a on to udělá — vytvoří web, napíše dokument, zanalyzuje data, nainstaluje program. Pracuje přímo s vašimi soubory na počítači (nebo v cloudu) a umí spouštět příkazy, hledat na webu a volat další nástroje.

Instalace

Nainstalujte Claude Code

Vyberte si cestu, která vám vyhovuje. Desktopová aplikace je nejjednodušší — stáhněte, nainstalujte, přihlaste se. Webová aplikace nepotřebuje vůbec nic instalovat. Terminál je pro ty, kdo chtějí plnou kontrolu.

  1. Otevřete claude.com/download v prohlížeči.
  2. Stáhněte instalátor pro macOS, Windows nebo Linux.
  3. Nainstalujte aplikaci — dvojklik na stažený soubor, odklikejte průvodce.
  4. Spusťte aplikaci a přihlaste se stejnými údaji jako na claude.com.
  5. V aplikaci klikněte na záložku „Code" — tím spustíte Claude Code.

Desktopová aplikace vypadá jako chat, ale navíc umí otevřít složky z vašeho počítače a pracovat s nimi. Přesně to budeme dělat v dalších krocích.

Pozor na „Select Folder": Při prvním spuštění relace je potřeba vybrat složku, ve které bude agent pracovat. Zvolte Local (ne Cloud) a pak vyberte konkrétní složku — třeba Plochu nebo Dokumenty. Pokud vidíte chybu „Select Folder First", ujistěte se, že jste klikli na název složky, ne jen otevřeli dialog.

Podrobný návod: Desktop quickstart (anglicky)

  1. Otevřete claude.ai/code v prohlížeči.
  2. Přihlaste se svým Claude účtem.
  3. Hotovo — žádná instalace. Stačí napsat první zprávu.

Omezení webové verze: Webová aplikace běží v cloudu, ne na vašem počítači. Primárně pracuje se soubory na GitHubu (lokální git repozitáře lze nahrát přes bundle). Pro většinu úkolů z tohoto průvodce to stačí, ale pro práci s lokálními soubory je lepší desktop nebo terminál.

Podrobný návod: Web quickstart (anglicky)

Nativní instalátor — nevyžaduje Node.js. 

curl -fsSL https://claude.ai/install.sh | bash

V PowerShellu:

irm https://claude.ai/install.ps1 | iex

Po instalaci napište claude v libovolné složce — při prvním spuštění se otevře prohlížeč k přihlášení.

Podrobný návod: CLI quickstart (anglicky)

První rozhovor

Napište první zprávu

Teď přijde ten nejlepší moment. Otevřete Claude Code (ať už v desktopové aplikaci, na webu, nebo v terminálu) a napište mu něco. Čímkoli.

Zkuste třeba tohle

Tohle je dobrý první prompt — agent vám ukáže, co všechno umí, a vy se zorientujete v tom, jak vypadá konverzace s ním.

Ahoj! Jsem tady poprvé. Ukaž mi v krátkosti, co umíš — a pak mi pomoz vytvořit jednoduchou osobní stránku s mým jménem a třemi věcmi, které mě baví.

Co se stane: Agent se představí, zeptá se na vaše jméno a tři věci — a pak vytvoří HTML soubor s vaší osobní stránkou. Můžete ji otevřít v prohlížeči a uvidíte výsledek. Trvá to asi minutu.

Gratulujeme — právě jste poprvé spolupracovali s AI agentem. To, co jste viděli, je základ všeho dalšího: vy popíšete, co chcete, a agent to vytvoří. V Level 1 se naučíte, jak agentovi dát víc kontextu, aby výsledky byly přesně takové, jaké potřebujete.

Časté otázky

Co se lidé nejčastěji ptají

Tyhle otázky padly na našem webináři od desítek účastníků. Odpovědi vám ušetří čas.

Agent ukládá soubory do složky, ve které jste ho spustili. V desktopové aplikaci je to složka, kterou jste vybrali na začátku relace. Pokud soubory nevidíte, řekněte agentovi: „Kde přesně jsi uložil ty soubory? Pošli mi celou cestu." — on vám ji vypíše a klidně soubor i otevře.

Lokální (Local) — agent běží na vašem počítači a pracuje přímo s vašimi soubory. Když něco vytvoří nebo změní, změny se projeví rovnou ve složce, kterou jste zvolili. Výpočty samotné probíhají na serverech Anthropic, ale soubory zůstávají u vás.

Cloud — agent běží v izolovaném prostředí na serverech Anthropic. Pracuje primárně s repozitáři na GitHubu — naklonuje je, provede změny a výsledek pošle zpět jako novou větev. Nic se nemění přímo na vašem disku. Hodí se, když chcete pracovat z prohlížeče bez instalace.

Pro začátek volte lokální — je to jednodušší a výsledky vidíte okamžitě u sebe na počítači.

Podrobněji: Claude Code on the web (anglicky)

Vaše prompty a soubory se posílají na servery Anthropic ke zpracování. U firemních plánů (Team, Enterprise) se data uchovávají maximálně 30 dní a nepoužívají se k trénování. U osobních plánů (Pro, Max) záleží na vašem nastavení soukromí v claude.ai/settings. Agent má přístup jen ke složce, ve které běží — ne k celému počítači. U důležitých akcí (mazání, instalace) se vás vždy zeptá na povolení.

Podrobněji: Security · Data usage (anglicky)

V terminálu zmáčkněte dvakrát Escape (nebo napište /rewind) — otevře se nabídka, kde můžete vrátit kód, konverzaci, nebo oboje na libovolný předchozí bod. V desktopové aplikaci najdete historii relací v postranním panelu. Více v Level 1.

Nejčastější důvod: spouštíte agenta v příliš velké složce bez kontextu. Agent pak musí prohledávat desítky souborů, aby pochopil, kde je. Řešení: vždy začněte ve složce konkrétního projektu a vytvořte si CLAUDE.md (více v Level 3) — jedna věta typu „tento projekt je osobní web v HTML" agentovi ušetří stovky zbytečných tokenů.

Společník průvodce

V našem repozitáři na GitHubu žije společník průvodce — AI pomocník v Claude Code, který zná obsah tohoto průvodce, má připravené prompty a umí odpovědět na časté otázky. Není povinný, ale může se hodit.

K jeho aktivaci potřebujete Git (nástroj na stahování projektů z GitHubu). Pokud ho nemáte, Claude Code vám s instalací pomůže.

Řekněte Claude Code

Jeden prompt, agent se o zbytek postará — nainstaluje Git (pokud chybí), naklonuje repozitář a ověří, že společník funguje.

Potřebuju si stáhnout materiály z repozitáře https://github.com/aibilitycz/claude-code-90-webinar.git — naklonuj ho do rozumného místa a pak ověř, že obsahuje složku .claude/skills/claude-code-guide. Pokud nemám Git, pomoz mi ho nejdřív nainstalovat.

Co očekávat: Agent naklonuje repozitář, ověří obsah a řekne vám, že společník je připravený. Při příštím spuštění Claude Code v této složce se automaticky aktivuje.
Něco se zaseklo?

Dva způsoby, jak se odseknout

Žádné chyby, které cestou potkáte, nejsou fatální. Máte dva způsoby, jak se dostat dál.

Zeptejte se Claude Code

Už máte nainstalovaného agenta, který čte chybové hlášky lépe než většina lidí. Popište mu, co jste zkusili a co vám to vypsalo. Většinou to sám rovnou napraví.

Jakýkoli jiný AI asistent

ChatGPT, Gemini, Copilot — stejný princip. Popište problém, přiložte chybovou hlášku. Moderní AI asistenti instalační problémy řeší spolehlivě.
