Pokročilé funkce

Pokročilé funkce

Funkce, které se hodí znát, jakmile se zorientujete v základech.

/rewind · Esc Esc

Rewind — vrácení změn

Dvakrát Esc nebo příkaz /rewind otevře nabídku se čtyřmi možnostmi:

  • Vrátit kód i konverzaci na vybraný bod
  • Vrátit pouze kód (konverzace zůstane)
  • Vrátit pouze konverzaci (kód zůstane)
  • Shrnout konverzaci od vybraného bodu

claude --continue · --resume

Resume — pokračování v chatu

claude --continue okamžitě navazuje na poslední konverzaci — ideální, když zavřete terminál a chcete pokračovat tam, kde jste skončili. claude --resume zobrazí seznam uložených konverzací a nechá vás vybrat kteroukoli z nich.

/config → remote control

Remote control — ovládání z telefonu

V terminálu zadejte /config a zapněte remote control. Pak otevřete Claude appku na mobilu a připojte se k běžící relaci. Spustíte práci na počítači, pak odejdete — a sledujete průběh nebo zadáváte pokyny z telefonu.

claude --worktree

Paralelní relace

claude --worktree vytvoří izolovanou kopii projektu (git worktree), ve které může agent pracovat souběžně s vaší hlavní větví. Hodí se, když chcete testovat experimentální změny, aniž by ovlivnily rozdělanou práci.

cat soubor.txt | claude -p "..."

Piping — jednorázové úkoly

Přepnutím vstupu přes rouru spustíte Claude Code pro jednorázový úkol bez otevírání interaktivní relace. Vhodné pro skripty a automatizace. Příklad: cat report.txt | claude -p "shrň klíčové body".

FAQ

Časté otázky

Agent má přístup jen ke složce, ve které běží — ne k celému počítači. U důležitých akcí (mazání, instalace) se vás vždy zeptá na povolení. Vaše prompty a soubory se posílají na servery Anthropic ke zpracování. U firemních plánů (Team, Enterprise) se data nepoužívají k trénování a uchovávají se maximálně 30 dní. U osobních plánů (Pro, Max) záleží na vašem nastavení soukromí v claude.ai/settings. Pro firmy Anthropic nabízí enterprise licencování s přísnějšími zárukami.

Podrobněji: Security · Data usage

Tokeny jsou jednotky „myšlení", které agent spotřebovává. Tři způsoby, jak je šetřit: 1) Začínejte ve správné složce — nespouštějte agenta v kořenovém adresáři. 2) Vytvořte si CLAUDE.md — jedna věta o projektu ušetří stovky tokenů. 3) Používejte /compact — zkrátí konverzaci a uvolní kontextové okno.

Podrobněji: Costs · Best practices

Lokální — agent pracuje přímo s vašimi soubory na počítači. Výpočty probíhají na serverech Anthropic, ale soubory zůstávají u vás. Cloud — agent běží v izolovaném prostředí na serverech Anthropic, pracuje primárně s repozitáři na GitHubu. Pro začátek volte lokální.

Podrobněji: Claude Code on the web

Skill je textový soubor s instrukcemi — říká agentovi, jak provést konkrétní úkol. Načítá se na vyžádání. Subagent je izolovaný pracovník s vlastním kontextovým oknem — agent si ho pouští na dílčí úkoly, aby nezaplnil svůj hlavní kontext. Plugin je balíček, který může obsahovat skills, subagenty, hooky a MCP servery dohromady.

Podrobněji: Skills · Features overview

Dvakrát Escape (nebo příkaz /rewind) otevře nabídku, kde vrátíte kód, konverzaci, nebo oboje na libovolný předchozí bod. Jedním Escape zastavíte agenta uprostřed práce. Nebo prostě řekněte: „Vrať to zpátky."

Efektivní práce

Tipy pro efektivní práci

Jednoduché návyky, které výrazně zlepší výsledky.

  • Začínejte ve správné složce

    Spouštějte agenta vždy ve složce projektu, ne v kořenovém adresáři. Agent pak čte jen relevantní soubory — méně tokenů, přesnější výsledky.

  • Vytvořte si CLAUDE.md

    Jeden soubor s popisem projektu, preferencemi a opakujícími se instrukcemi. Agent ho čte automaticky při každém spuštění. Postup najdete v Level 3.

  • Používejte /clear mezi nesouvisejícími úkoly

    Kontext z předchozího úkolu může plést agenta při novém. /clear smaže konverzaci a nechá soubory beze změny — čistý start bez zbytečného šumu.

  • Buďte konkrétní v promptech

    Čím přesnější zadání, tím lepší výsledek. Místo „vytvoř web" zadejte „vytvoř landing page pro fitness studio s oddíly: hero, ceník, kontakt". Kontext a záměr jsou klíčové.

  • Nechte agenta ověřovat svou práci

    Na konec zadání přidejte „zkontroluj si to po sobě" nebo „ověř, že vše funguje". Agent si projde výsledek vlastním pohledem a zachytí chyby dřív, než je uvidíte vy.
