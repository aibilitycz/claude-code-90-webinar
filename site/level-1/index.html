Level 1

Váš první projekt s Claude Code

Teď to začne být zajímavé. Postavíte něco skutečného a přitom se naučíte nejdůležitější věc o práci s AI agentem: čím víc kontextu mu dáte, tím lepší výsledek dostanete.

Asi 20 minut Potřebujete hotový Začátek

Co se naučíte

  • Proč holý prompt dává průměrný výsledek — a jak ho vylepšit kontextem
  • Jak agentovi dát soubory, obrázky a informace z webu
  • Jak funguje cyklus: přemýšlení → tvorba → oprava → povolení → výsledek
  • Jak použít plánování (plan mode) pro složitější úkoly
Krok 1 ze 4

Začněte bez kontextu

Nejdřív si vyzkoušejte, co se stane, když agentovi řeknete, co chcete, ale nedáte mu žádný kontext. Výsledek bude funkční, ale obecný — a to je přesně ten bod, od kterého budeme vylepšovat.

Prompt 1

Otevřete Claude Code v prázdné složce a napište:

Vytvoř mi osobní stránku — bio, kontakt, portfolio. Čistý, moderní design.

Co uvidíte: Agent vytvoří HTML soubor s generickým obsahem (Lorem ipsum, placeholder jméno). Funkční, ale ne vaše.

Klíčové zjištění: Bez kontextu agent hádá. Výsledek je „pro kohokoli" — a proto pro nikoho konkrétně. Přesně tohle teď napravíme.

Krok 2 ze 4

Přidejte kontext

Teď dáme agentovi něco, z čeho může čerpat. Vytvořte v té samé složce soubor s pár informacemi o sobě (nebo o komkoli, pro koho stránku děláte).

Připravte si soubor

Vytvořte soubor muj-obsah.md a napište do něj pár vět o sobě — jméno, čím se zabýváte, co vás baví, kontakt. Stačí pár řádků, nemusí to být dokonalé. Nebo řekněte agentovi, ať soubor vytvoří za vás:

Prompt 2

Teď agentovi řeknete, aby pracoval s vaším obsahem:

Přečti si soubor muj-obsah.md a podle něj přepracuj tu osobní stránku. Použij skutečné informace, žádné placeholdery.

Co uvidíte: Agent přečte váš soubor a vytvoří stránku s vaším skutečným jménem, vaším popisem, vašimi informacemi. Zásadní rozdíl oproti prvnímu pokusu.

Princip: „Kontextu je král." Agent neumí hádat, co chcete. Ale když mu dáte materiál — soubor, odkaz, obrázek — pracuje s ním přesně a věrně. Čím víc kontextu, tím lepší výsledek.

Krok 3 ze 4

Přidejte inspiraci

Claude Code umí víc než číst text. Ukažte mu obrázek jako designovou inspiraci a pošlete ho hledat na web.

Vizuální inspirace

Uložte si do složky screenshot webu, který se vám líbí (nebo jakýkoli obrázek s designem, který chcete napodobit).

Prompt 3

Agent vidí obrázky a umí z nich čerpat styl:

Podívej se na obrázek inspirace.png — líbí se mi tenhle vizuální styl. Uprav moji stránku tak, aby měla podobnou estetiku. Taky prosím vyhledej na webu aktuální trendy v osobních stránkách na rok 2026 a zapracuj je.

Co uvidíte: Agent analyzuje obrázek, vyhledá na webu trendy, a přepracuje design vaší stránky. Výsledek má styl inspirace a moderní prvky.

Agent vidí obrázky. Můžete mu ukázat screenshot, wireframe, nákres na papíře — cokoli vizuálního. Taky umí hledat na webu aktuální informace, takže nepracuje jen s tím, co „zná" ze svého tréninku.

Krok 4 ze 4

Plánujte, pak stavte

Pro složitější úkoly má Claude Code speciální režim: nejdřív naplánuje, co udělá, a teprve pak to provede. Tím předejdete situacím, kdy agent jde špatným směrem a vy to zjistíte pozdě.

Prompt 4

Požádejte agenta o plán a schvalte ho:

Chci přidat na stránku sekci „Projekty" se třemi kartami. Nejdřív mi navrhni plán, jak to uděláš, a počkej na moje schválení, než začneš.

Co uvidíte: Agent popíše svůj plán (jaké sekce přidá, jak upraví layout, jaký styl použije) a počká na vaše „OK" nebo úpravy. Až schválíte, provede změny.

Princip plánování: U větších úkolů vždycky nejdřív plán, pak realizace. Ušetříte čas, který byste jinak strávili opravováním výsledku. Můžete přímo napsat /plan nebo v terminálu přepínat režimy pomocí Shift+Tab.

Pod kapotou

Jak agent přemýšlí

Všimněte si, co se děje, když agent pracuje. Není to jen „otázka → odpověď" jako u chatbota. Agent prochází cyklem:

Přemýšlení Analyzuje úkol
Tvorba Píše kód, soubory
Oprava Kontroluje sám sebe
Povolení Ptá se vás
Výsledek Hotovo

Tohle je zásadní rozdíl oproti chatbotovi. Chatbot odpovídá. Agent pracuje — čte soubory, spouští příkazy, kontroluje výsledky, opravuje chyby, a u důležitých kroků se vás ptá na souhlas. Vy jste šéf, on je schopný asistent.

Záchranná síť

Když se něco nepovede

Agent občas udělá něco jinak, než jste chtěli. Nemusíte panikařit — všechno se dá vrátit.

Esc Esc V terminálu zmáčkněte dvakrát Escape (nebo napište /rewind) — otevře se nabídka, kde vrátíte kód, konverzaci, nebo oboje na libovolný předchozí bod.
Esc Jedním Escape zastavíte agenta uprostřed práce — když vidíte, že jde špatným směrem.
Prompt Nebo prostě řekněte: „Vrať to zpátky, jak to bylo před poslední změnou." — agent umí vrátit vlastní úpravy.

Nebojte se experimentovat. Vždycky se dá vrátit zpátky.

Zkuste to sami

Tři nápady na procvičení

Vyberte si jeden a pusťte se do toho. Každý z nich využívá to, co jste se právě naučili — kontext, soubory, plánování.

A

Osobní stránka

Vytvořte si svou vlastní stránku od nuly. Připravte soubor s informacemi o sobě a obrázek jako inspiraci.

„Vytvoř mi osobní stránku podle souboru muj-obsah.md a podle vizuálního stylu v obrázku inspirace.png. Stránka má být v češtině."

B

To-do aplikace

Nechte agenta vytvořit jednoduchou aplikaci na úkoly. Funguje rovnou v prohlížeči.

„Vytvoř mi jednoduchou to-do aplikaci v HTML, CSS a JavaScriptu. Chci přidávat úkoly, označovat jako hotové a mazat. Pěkný design, tmavý režim."

C

Stránka o čemkoli

Máte koníček, firmu, projekt? Nechte agenta vytvořit stránku o tom, co vás baví.

„Vyhledej na webu informace o [vaše téma] a vytvoř krásnou informační stránku v češtině. Moderní design, s obrázky z webu."
