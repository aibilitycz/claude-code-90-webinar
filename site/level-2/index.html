Level 2

Skills a pluginy

V Level 1 jste s Claude Code něco postavili. Teď to samé uděláte znovu — a pak z toho vytvoříte skill, aby to příště šlo na jedno slovo.

Asi 15 minut Rozšiřuje Level 1
Princip

OHIO — „Only Handle It Once"

Vyřeš to napoprvé, pak z toho udělej skill — a příště to nedělej vůbec. Ten skill to udělá za tebe.

Simon Willison (autor Djanga, AI researcher): „Claude Skills jsou možná větší věc než MCP."

Krok 1

Udělejte úkol ručně

Nejdřív si s Claude Code projdete celý úkol krok za krokem — jako byste ho dělali poprvé. Například: vytvořte LinkedIn carousel ze tří screenshotů.

Zadejte úkol

Otevřete Claude Code v libovolné složce a napište:

Tady máš tři screenshoty (screen1.png, screen2.png, screen3.png). Udělej z nich LinkedIn carousel jako PDF — jeden screenshot na stránku, čistý layout, připraveno k publikování.

Co uvidíte: Claude Code přečte obrázky, sestaví layout a vygeneruje hotové PDF. Celý proces trvá pod minutu.

Princip: Poprvé to děláte ručně — díváte se, jak Claude Code postupuje, a ověřujete, že výsledek sedí. Tohle je důležité: skill vzniká z ověřeného postupu, ne z prvního pokusu.

Krok 2

Udělejte z toho skill

Výsledek sedí? Řekněte Claude Code, aby z toho, co právě udělal, vytvořil skill. Nemusíte nic psát ručně — agent to napíše sám.

Nechte agenta vytvořit skill

Ve stejné session napište:

To, co jsme právě udělali, chci opakovat. Vytvoř z toho skill — SKILL.md soubor, který můžu příště zavolat jedním příkazem.

Co uvidíte: Claude Code vytvoří soubor SKILL.md s hlavičkou (jméno, popis, spouštěcí fráze) a tělem (instrukce pro agenta). Pár desítek řádků Markdownu — žádná magie.

Princip: Skill je textový soubor v Markdownu, který agentovi říká: „Když se tě někdo zeptá na X, udělej Y tímhle způsobem." Může obsahovat instrukce, šablony, pravidla, příklady.

Krok 3

Použijte skill

Skill existuje. Teď ho vyzkoušejte — otevřete novou session a zadejte úkol jednou větou. Claude Code skill rozpozná sám a spustí ho.

Vyvolejte skill

V nové session napište jednu větu:

Udělej LinkedIn carousel z těchhle screenshotů.

Co uvidíte: Claude Code rozpozná, že pro tenhle úkol existuje skill, a provede ho automaticky — se stejným postupem, stejnou kvalitou, bez dalšího vysvětlování.

OHIO v akci: Poprvé jste úkol řešili ručně. Podruhé jste řekli jednu větu. Potřetí, počtvrté, podeváté — pořád ta jedna věta. Složené úročení vaší vlastní práce.

Krok 4

Prozkoumejte marketplace

Nemusíte si všechno stavět sami. V Claude Code je zabudovaný marketplace s tisícovkami hotových pluginů od komunity.

Otevřete katalog

V Claude Code napište: 

/plugin

Zobrazí se nabídka. Vyberte Discover.

Co uvidíte: Katalog s kategoriemi — obrázky, audio, deploy, databáze, design, komunikace. Reálné pluginy od reálných týmů. Scrollujte a podívejte se, co je k dispozici.

Princip: Skills jsou vaše vlastní postupy. Pluginy jsou rozšíření od komunity — hotové nástroje, které agentovi přidávají schopnosti, které sám o sobě nemá.

Krok 5

Nainstalujte plugin

Vybrali jste si plugin v katalogu? Instalace je jeden příkaz.

Z katalogu Discover

V katalogu (z předchozího kroku) vyberte plugin, který vás zaujal. Claude Code ho nainstaluje sám — potvrdíte a je to.

Z GitHubu

Pokud víte o konkrétním pluginu na GitHubu, nainstalujte ho přímo: 

/plugin marketplace add ondrej-svec/heart-of-gold-toolkit
Co uvidíte: Claude Code stáhne plugin a zpřístupní jeho skills ve vaší session. Od teď máte k dispozici nové příkazy a postupy.

Dobrá zpráva

SKILL.md je otevřený standard

Skills nejsou zamčené v Claude Code. Formát SKILL.md je otevřený — funguje i v Codex CLI od OpenAI, v Cursor, v Gemini CLI a v dalších nástrojích. Neinvestujete do jednoho nástroje, investujete do přenositelného formátu.

Zkuste to sami

Tři věci k vyzkoušení

Vyberte si jednu, dvě, nebo všechny tři:

A) Vytvořte si vlastní skill

Udělejte s agentem libovolný opakovaný úkol a pak z něj vytvořte skill:

Napiš mi LinkedIn post o [vaše téma]. Pak z toho postupu udělej skill, který příště zavolám jedním příkazem.

B) Prozkoumejte marketplace

Podívejte se, co všechno je k dispozici:

/plugin

C) Nainstalujte plugin z GitHubu

Vyzkoušejte instalaci z příkazové řádky:

/plugin marketplace add ondrej-svec/heart-of-gold-toolkit

Další krok: Level 3

Umíte stavět projekty a rozšiřovat agenta o nové schopnosti. V Level 3 se naučíte, jak z Claude Code udělat platformu — CLAUDE.md, MCP servery a orchestrace.
