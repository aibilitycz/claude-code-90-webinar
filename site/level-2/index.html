Level 2

Skills a pluginy

V Level 1 jste postavili projekt ručně. Teď se naučíte, jak opakovanou práci vyřešit jednou a uložit — aby se příště stačilo říct jedno slovo.

Asi 15 minut Rozšiřuje Level 1

Co se naučíte

  • Co je to skill a proč je to silnější než prompt
  • Jak jedním příkazem rozšířit schopnosti agenta o tisíce nových funkcí
  • Kde najít pluginy a skills od komunity
  • Jak si nechat vytvořit vlastní skill
Princip

OHIO — „Only Handle It Once"

Každou opakovanou práci vyřešte jednou. Pak ji uložte jako skill — krátký textový soubor, který agentovi říká, jak daný úkol provést. Příště stačí zavolat jméno skillu a agent to zopakuje přesně, rychle a konzistentně.

Simon Willison (AI researcher): „Claude Skills are maybe a bigger deal than MCP." — Skills jsou možná důležitější než MCP, protože mění způsob, jakým lidé s AI agenty pracují každý den.

Skills

Co je skill

Skill je textový soubor v Markdownu (SKILL.md), který agentovi říká: „Když se tě někdo zeptá na X, udělej Y tímhle způsobem." Může obsahovat instrukce, šablony, pravidla, příklady — cokoli, co agent potřebuje k tomu, aby úkol zvládl sám.

Příklad z praxe: denní brief

Představte si, že každé ráno potřebujete souhrn novinek z vašeho oboru. Bez skillu to vypadá takhle: otevřete agenta, napíšete dlouhý prompt s instrukcemi, počkáte, výsledek ručně upravíte. Se skillem řeknete /daily-brief — a agent udělá přesně to samé, pokaždé stejně, každý den.

Jak si nechat vytvořit skill

Nemusíte skill psát ručně. Udělejte úkol jednou s agentem, a pak mu řekněte:

Nechte agenta vytvořit skill

To, co jsme právě udělali, chci opakovat. Vytvoř z toho skill — SKILL.md soubor, který můžu příště zavolat jedním příkazem.

Co uvidíte: Agent vytvoří soubor SKILL.md s instrukcemi, které zachycují váš pracovní postup. Příště stačí zavolat skill jménem.
Pluginy

Marketplace: tisíce rozšíření

Skills jsou vaše vlastní pracovní postupy. Pluginy jsou rozšíření od komunity — tisíce hotových nástrojů, které agentovi přidávají nové schopnosti: od generování obrázků přes správu e-mailů po nasazení webu.

Jak nainstalovat plugin

Napište v Claude Code /plugin a vyberte Discover. Otevře se katalog s kategoriemi — vyberete, co chcete, a agent si plugin sám nainstaluje.

Zkuste to

Tenhle příkaz vám ukáže, co je k dispozici:

/plugin

Co pluginy umí

S pluginy agent zvládne věci, které sám o sobě neumí:

Obrázky — generování a editace
Audio — text-to-speech, podcasty
Deploy — nasazení webu na internet
Databáze — připojení k datům
Design — Figma, prototypování
Komunikace — Slack, e-mail, Notion
Kde hledat

Zkuste to sami

Dvě věci k vyzkoušení

Vyberte si jednu nebo obě:

A) Prozkoumejte marketplace

Podívejte se, jaké pluginy jsou k dispozici:

/plugin

B) Vytvořte si vlastní skill

Udělejte s agentem nějaký opakovaný úkol a pak ho uložte:

Vytvoř mi LinkedIn post o [vaše téma]. Pak z toho postupu udělej skill, který příště zavolám jedním příkazem.

Dobrá zpráva

SKILL.md je otevřený standard

Skills nejsou zamčené v Claude Code. Formát SKILL.md je otevřený — funguje i v Codex CLI od OpenAI, v Cursor, v Gemini CLI a v dalších nástrojích. Co vytvoříte jednou, použijete kdekoli.
