Level 3

Orchestrace

Claude Code přestane být chatbot a stane se platformou. CLAUDE.md mu řekne, kdo jste. MCP servery ho propojí s vašimi nástroji. Modely vám dají kontrolu nad rychlostí a kvalitou. Všechno dohromady — to je orchestrace.

Asi 15 minut Pro pokročilé

Co se naučíte

  • Co je CLAUDE.md a proč zásadně snižuje halucinace
  • Co jsou MCP servery a jak propojí agenta s vnějším světem
  • Jak přepínat mezi modely (Haiku, Sonnet, Opus)
  • Jak všechno spojit dohromady do jednoho pracovního systému
CLAUDE.md

Řekněte agentovi, kdo jste

CLAUDE.md je textový soubor, který agent přečte pokaždé, když ho spustíte. Obsahuje pravidla, preference a kontext — všechno, co potřebuje vědět, aby pracoval tak, jak chcete vy. Je to ten nejsilnější nástroj proti halucinacím.

Dva typy CLAUDE.md

Projektový (./CLAUDE.md) — pravidla pro konkrétní projekt. Například: „Piš vždycky česky" nebo „Používej tenhle framework." Sdílí se s týmem přes Git.

Osobní (~/.claude/CLAUDE.md) — vaše globální preference. Například: „Jsem marketér, ne programátor" nebo „Na služební cesty létám s příručním zavazadlem."

Lokální (./CLAUDE.local.md) — osobní poznámky k projektu, které se nesdílí s týmem. Přidejte si ho do .gitignore.

Klíčový princip: „Agent musí číst, ne hádat." Jedna věta v CLAUDE.md dramaticky snižuje halucinace. Místo hádání agent vychází z toho, co jste mu napsali.

Zkuste to

Nechte agenta vytvořit váš osobní CLAUDE.md:

Vytvoř mi osobní CLAUDE.md v ~/.claude/CLAUDE.md. Jsem [vaše profese], pracuji v [obor/firma], preferuji [jazyk, styl, formát]. Přidej pravidla, která mi budou užitečná při každém spuštění.

MCP

Propojte agenta s vnějším světem

MCP (Model Context Protocol) je způsob, jak agenta propojit s externími službami — kalendáři, databázemi, API, vyhledávači letenek. MCP servery jsou jako adaptéry: každý z nich přidává agentovi přístup k jednomu nástroji.

Příklad: hledání letenek

Nainstalujte MCP server pro vyhledávání letenek (třeba Kiwi) a pak řekněte agentovi:

Příklad s MCP

Najdi mi letenky z Prahy do Lisabonu na příští víkend, jednosměrku, co nejlevnější. Pak mi to přidej do Google kalendáře.

Co se stane: Agent zavolá Kiwi MCP server (letenky), najde možnosti, a zavolá Google Calendar MCP (kalendář) — vytvoří událost. Dvě služby, jeden prompt.

Existují stovky veřejných MCP serverů — pro databáze, cloud služby, komunikaci, projektové řízení a víc. Každý z nich se instaluje jedním příkazem přes /plugin.

Modely

Tři modely, tři rychlosti

Claude Code umí přepínat mezi třemi modely. Každý je jinak rychlý a jinak schopný. Přepínáte příkazem /model.

Haiku

Nejrychlejší, nejlevnější. Pro jednoduché úkoly, rychlé odpovědi, triviální úpravy.

Kdy: „přejmenuj proměnnou", „vypiš seznam"

Sonnet

Vybalancovaný. Dobrý poměr rychlosti a kvality pro většinu každodenní práce.

Kdy: „napiš blog post", „analyzuj data"

Opus

Nejschopnější. Pro složité úkoly, strategické rozhodování, komplexní analýzy.

Kdy: „navrhni architekturu", „zrecenzuj celý projekt"

Bezpečnost

Jak je to s přístupem a daty

Častá otázka: může agent smazat moje soubory? Vidí moje hesla? Kam posílá moje data?

Přístup jen ke složce

Agent má přístup jen ke složce, ve které běží — ne k celému počítači. U nebezpečných akcí (mazání, instalace) se vás vždy zeptá na povolení.

Kontrola nad daty

Vaše prompty a soubory se posílají na servery Anthropic ke zpracování. U firemních plánů (Team, Enterprise) se data nepoužívají k trénování a uchovávají se maximálně 30 dní. U osobních plánů si nastavení soukromí upravíte v claude.ai/settings.

Režimy oprávnění

Můžete si zvolit, jak moc volnosti agentovi dáte: od „ptej se na všechno" po „dělej sám". Pro začátek nechte výchozí nastavení — agent se bude ptát u důležitých kroků.

Pro firmy Anthropic nabízí enterprise licencování s přísnějšími bezpečnostními zárukami.

Tip

Jak šetřit tokeny

Tokeny jsou jednotky „myšlení", které agent spotřebovává. Čím víc musí hledat a přemýšlet, tím víc jich utratí. Tři způsoby, jak je šetřit:

1. Začínejte ve správné složce. Nespouštějte agenta v kořenové složce celého počítače. Vždy jděte do konkrétního projektu.
2. Vytvořte si CLAUDE.md. Jedna věta o projektu ušetří stovky tokenů, protože agent nemusí hádat, s čím pracuje.
3. Používejte /compact. Když konverzace naroste, tenhle příkaz ji zkrátí a uvolní místo v kontextovém okně.
Zlaté pravidlo

Agent musí číst, ne hádat

Tohle je nejdůležitější pravidlo celého průvodce. Kdykoli se vám zdá, že agent „halucinuje" nebo dává špatné výsledky, je řešení téměř vždy stejné: dejte mu víc kontextu.

1. CLAUDE.md — Napište, kdo jste a co chcete. Agent to přečte pokaždé.
2. Soubory — Dejte agentovi materiál. Ať čte vaše texty, data, obrázky.
3. Web search — Nechte ho hledat aktuální informace, ne hádat z tréninku.

Čím víc kontextu = tím míň halucinací. Není to magie — je to prostě rozdíl mezi hádáním a čtením.

Všechno dohromady

Orchestr pod jednou taktovkou

Představte si orchestr. CLAUDE.md je partitura. Skills jsou nástroje. MCP servery jsou hudebníci. Modely určují tempo. A Claude Code je dirigent, který to všechno spojuje dohromady.

Průvodce dokončen
