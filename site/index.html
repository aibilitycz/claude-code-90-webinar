Nainstalujte Claude Code

Claude Code běží na víc místech — v desktopové aplikaci, v prohlížeči, v terminálu, na mobilu nebo v editoru. Pro webinář potřebujete cestu, která umí otevřít naklonovanou složku z vašeho počítače. Doporučujeme desktopovou aplikaci — je to nejjednodušší start. Pokud chcete vidět přesně to, co my během živého vysílání, zvolte terminál.

Webová aplikace claude.ai/code funguje pro běžný chat s Claude, ale neumí načíst společníka webináře — ten žije v naklonované složce na vašem disku. Proto pro webinář volte desktop nebo terminál.

  1. Otevřete claude.com/download v prohlížeči.
  2. Stáhněte instalátor pro macOS nebo Windows.
  3. Nainstalujte aplikaci stejně jako jakoukoli jinou — dvojklik na stažený soubor a odklikejte průvodce.
  4. Spusťte aplikaci a přihlaste se stejnými údaji jako u claude.com.

Aplikace vypadá jako chat (podobně jako ChatGPT), ale navíc umí otevřít složky z vašeho počítače a pracovat s nimi. Tuhle vlastnost budeme během webináře hodně používat.

Doporučený způsob: nativní instalátor, který nevyžaduje Node.js. 

curl -fsSL https://claude.ai/install.sh | bash

V PowerShellu (ne v CMD):

irm https://claude.ai/install.ps1 | iex

Po instalaci spusťte příkaz claude v libovolné složce — při prvním spuštění se otevře prohlížeč k přihlášení.

Nainstalujte Git

Git je nástroj, kterým se stahují projekty z GitHubu. Dnes ho potřebujete jen na jednu věc — stáhnout náš repozitář se společníkem webináře.

V Kroku 2 jste si nainstalovali Claude Code — teď ho pojďme využít. Otevřete ho a pošlete mu tuhle zprávu:

Potřebuju nainstalovat Git na svůj počítač. Zjisti, jaký mám operační systém, proveď mě instalací krok za krokem a na konci ověř, že to funguje.

Agent zjistí, co máte za systém, provede vás instalací a na konci ověří, že je všechno v pořádku. Když se cestou na něčem zadrhne, zeptá se vás.

Vyberte si podle toho, jaký máte operační systém:

Na macOS už možná Git máte. Otevřete Terminal (Cmd + Space → napsat „Terminal") a zkuste:

git --version
Co očekávat: něco jako git version 2.x.x. Pokud to vidíte, máte hotovo.

Pokud vám macOS místo toho nabídne instalaci Xcode Command Line Tools, klikněte na „Install" — stáhne se všechno, co potřebujete (pár minut).

  1. Otevřete git-scm.com/download/win.
  2. Stáhněte a nainstalujte 64-bit Git for Windows.
  3. Během instalace klikejte „Next" a nechte všechno, jak je — výchozí nastavení stačí.
  4. Po dokončení instalace otevřete PowerShell a ověřte instalaci:
git --version
Co očekávat: něco jako git version 2.x.x.windows.x.

Podle distribuce Linuxu:

# Debian, Ubuntu
sudo apt install git

# Fedora, RHEL
sudo dnf install git

# Arch
sudo pacman -S git
Ověření: git --version by mělo vrátit git version 2.x.x.

Stáhněte materiály k webináři

Teď si stáhneme náš repozitář — dostanete tím společníka webináře (pomocníka, který vám během živého vysílání pomáhá), materiály ke každé části webináře a cheat sheet pro první půlhodinu po něm.

Otevřete Claude Code a pošlete mu tuhle zprávu. On najde, kam materiály uložit, a postará se o zbytek.

Stáhni mi materiály k webináři Intro do Claude Code. Naklonuj repozitář https://github.com/aibilitycz/claude-code-90-webinar.git do rozumného místa (třeba složky Projects v mém domovském adresáři) a pak mi ukaž, co jsem dostal — hlavně jestli obsahuje složku .claude/skills/claude-code-guide.

Otevřete terminál (macOS/Linux) nebo PowerShell (Windows) a spusťte:

git clone https://github.com/aibilitycz/claude-code-90-webinar.git
cd claude-code-90-webinar
Co očekávat: Git vypíše něco jako Cloning into 'claude-code-90-webinar'… a za pár vteřin jste ve vlastní kopii repozitáře.

Krok 5 ze 5

Preflight check

Posledních pár minut. Ověřte si, že všechno funguje — ať nás 16. dubna ráno nic nepřekvapí.

Otevřete Claude Code v naklonované složce a pošlete mu tuhle jednu zprávu. Projde všechny kontroly za vás a na konci vám řekne, jestli je všechno v pořádku.

Jsem účastník webináře Intro do Claude Code. Zkontroluj mi prosím, že mám všechno připravené. Ověř: 1) funguje Git, 2) jsem ve správné složce s materiály webináře, 3) máš přístup ke společníkovi webináře (skill v .claude/skills/claude-code-guide/). Pak zkus na sobě samém zavolat společníka tak, že řekneš „Co mám zkusit poprvé?". Na konci mi krátce shrň, jestli jsem připraven/a.

Tři rychlé kontroly v terminálu / PowerShellu. Ujistěte se, že jste ve složce claude-code-90-webinar, kterou jste si naklonovali v Kroku 4.

1. Git funguje

git --version
Chceme vidět: git version 2.x.x (jakákoli verze 2 stačí).

2. Jste ve správné složce

ls .claude/skills
Chceme vidět: claude-code-guide. Pokud vám ls nefunguje (Windows PowerShell), použijte dir .claude\skills.

3. Claude Code se ohlásí

Spusťte Claude Code v téhle složce (ať už v terminálu příkazem claude, nebo v desktopové aplikaci volbou „Open folder") a do chatu napište:

Co mám zkusit poprvé?

Společník by se měl ozvat s nabídkou tří možností. Pokud ano, jste připraveni. Pokud vám místo toho Claude odpoví obecně („o webináři nic nevím"), nejste ve správné složce nebo se skill nenačetl — zkuste to znovu po cd claude-code-90-webinar.

Zeptejte se Claude Code

Už jste v Kroku 2 nainstalovali agenta, který umí číst chybové hlášky lépe než většina lidí. Otevřete Claude Code a pošlete mu přesně to, co jste zkusili a co vám to vypsalo. Agent vám poradí, co dál — a většinou to sám napraví.
