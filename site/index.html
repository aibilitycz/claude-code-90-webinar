Naučte se Claude Code

AI agent, který pracuje s vašimi soubory, hledá na webu a staví věci za vás — od osobní stránky po analýzu dat. Tenhle průvodce vás provede krok za krokem, vlastním tempem, bez jediného řádku kódu.

Rozdíl

Chatbot vs. Agent — v čem je to jiné

Claude Code není ChatGPT. Chatbot odpovídá na otázky. Agent plní úkoly — čte soubory, spouští příkazy, opravuje chyby, ptá se na souhlas. Tady je rozdíl v jedné tabulce.

Oficiální materiály od Anthropic

Kromě tohoto průvodce máte k dispozici bezplatné kurzy, dokumentaci a komunitu přímo od tvůrců Claude.

Claude Code 101

Bezplatný videokurz od Anthropic. Od prvního spuštění po každodenní práci. Anglicky, ale s ukázkami kódu, kterým rozumíte i bez angličtiny.

Oficiální dokumentace

Kompletní referenční příručka na code.claude.com. Quickstarty pro desktop, web i terminál, best practices, běžné scénáře.

Tipy a triky

Příkazy, klávesové zkratky, pokročilé funkce (rewind, resume, remote control) a FAQ. Tahák pro každodenní práci.
